Portafolio de Simulaciones

Resumen Tema 1: Integradores

Este tema introduce los fundamentos matemáticos y físicos necesarios para la simulación por ordenador. Se explica cómo representar el movimiento mediante funciones y ecuaciones paramétricas, así como la importancia del uso de vectores para representar magnitudes físicas como posición y velocidad. Se profundiza en las ecuaciones diferenciales ordinarias (EDO) que modelan el cambio en el tiempo y se estudian diferentes métodos de integración numérica para resolverlas, como el método de Euler, Heun y Runge-Kutta. Finalmente, se aplica este conocimiento a simulaciones gráficas básicas usando ciclos de simulación que actualizan la posición y velocidad de objetos en movimiento.

Simulación Bubble Shooter (Método de Euler)

Objetivo

Simular el disparo de una bola verde hacia un conjunto de bolas rojas estáticas usando Processing. El usuario elige la dirección con el ratón, y la bola se mueve en línea recta con velocidad constante.

Fundamentos Aplicados

  • Vectores: Representación de posición y velocidad con PVector
  • Movimiento sin fuerzas: Trayectoria uniforme a velocidad constante
  • Integrador de Euler Explícito: Para actualizar la posición

Explicación

Al hacer clic con el ratón (mouseClicked), se calcula la dirección del disparo restando la posición inicial de la bola y la posición del puntero. Esta dirección se normaliza y se multiplica por una velocidad constante v_bola para obtener el vector velocidad _v.

En cada fotograma (draw()), se llama a updateSimulation(), donde se aplica el método de Euler: 

_s += _v * dt

Esta ecuación se basa en: 

s(t+Δt) ≈ s(t) + v(t) * Δt

Se omiten fuerzas como la gravedad o fricción, por lo tanto el movimiento es lineal y uniforme.

Vídeo de la Simulación

Tema 2: Sistemas de Partículas

En este tema se estudian los sistemas de partículas como elementos básicos de simulación física. Se comienza con la partícula solitaria, idealización matemática de objetos sin estructura interna, y se simulan casos como partículas flotantes, considerando fuerzas como la flotación y la estabilidad. A continuación, se abordan sistemas de partículas más complejos, como emisores de humo, fuego o salpicaduras, donde cada partícula tiene una vida limitada y responde a fuerzas como la gravedad o el viento.

Se tratan también las colisiones, tanto entre partículas como entre partículas y planos. Se cubren sus tres fases: detección, reposicionamiento y respuesta física. Para aumentar la eficiencia, se aplican estructuras de datos como grids 2D/3D y tablas hash que permiten detectar colisiones e interacciones locales sin recorrer todas las partículas, lo que es clave en sistemas de gran escala. Finalmente, se presenta el concepto de interacción local para simular comportamientos como bandadas de aves o multitudes, donde solo se consideran las partículas cercanas.

Tema 3: Objetos Deformables

En este bloque se exploran simulaciones avanzadas basadas en modelos físicos como masa-muelle y propagación de ondas. Se implementan animaciones complejas de objetos deformables como banderas, pelos y superficies acuáticas. Cada proyecto se centra en cómo fuerzas como la gravedad, el viento, la amortiguación o la interacción con el usuario afectan dinámicamente a sistemas compuestos por partículas conectadas. Se aplican distintas estrategias de conectividad y funciones matemáticas para lograr comportamientos realistas y visualmente atractivos.

Banderas

Esta simulación muestra tres banderas animadas en 3D utilizando el modelo físico masa-muelle. Cada bandera es una malla rectangular compuesta por partículas (nodos) unidas por muelles. El sistema responde a fuerzas como la gravedad, un viento con variaciones suaves (aleatorias en X y también en Y), y amortiguación, para simular de forma realista el movimiento de una tela al viento.

Se comparan tres variantes del sistema:

  • Structured: solo conecta partículas con sus vecinas vertical y horizontalmente.
  • Structured + Shear: añade conexiones diagonales para mayor realismo.
  • Structured + Shear + Bend: agrega conexiones de flexión (dobles saltos), lo que simula mejor la resistencia a curvatura.

Visualmente, las banderas son rojas, el fondo blanco, y están montadas sobre mástiles negros.

Pelos

Este proyecto simula una gran cantidad de pelos o cuerdas colgando, también mediante un modelo masa-muelle en 2D. Cada pelo está compuesto por varios puntos conectados por muelles, con un extremo fijo.

Las fuerzas que actúan incluyen gravedad, amortiguamiento para evitar oscilaciones infinitas, y la interacción con el ratón (arrastre directo).

Interacción del usuario:

  • C: cambia los colores de los pelos aleatoriamente.
  • N: los devuelve a color negro.
  • R: reinicia la simulación con nuevos pelos aleatorios.

Ondas

Esta simulación representa una superficie de agua dinámica en 3D basada en una cuadrícula tridimensional de puntos que se deforman usando tres funciones de onda: Gerstner, Radial y Direccional.

Cada vértice de la malla simula el comportamiento del agua mediante el ajuste continuo de su altura, desplazamientos horizontales y cálculo de normales. El surfista animado, posado sobre la malla, adapta su orientación según la inclinación de la superficie gracias a la normal local.

Las funciones de onda:

  • Gerstner: función no lineal que simula con realismo crestas, desplazamientos verticales y horizontales, útil para simular el mar.
  • Radial: genera perturbaciones circulares desde un centro fijo, ideales para simular gotas u ondas de impacto.
  • Direccional: simula oleaje constante en una dirección usando una sinusoide lineal.

Mediante las teclas 1, 2 y 3 se puede alternar entre las funciones activas. Los parámetros como spacing, amplitude, wavelength y waveSpeed permiten ajustar resolución y comportamiento de la ola. TIME_ACCEL acelera o ralentiza la animación, y TS regula el paso temporal de integración, afectando la precisión y estabilidad.

Visualmente, se emplea un degradado entre azul profundo y turquesa, y sombreado realista gracias al uso de normales por vértice, generando brillos y sombras suaves que refuerzan la sensación de profundidad y movimiento del agua.
